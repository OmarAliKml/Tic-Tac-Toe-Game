# Tic Tac Toe 🎮

A sleek and Tic Tac Toe game built with Flutter, featuring smooth animations, AI opponent, and local multiplayer.

## ✨ Features

- 🎯 Single Player vs AI
- 👥 Local Two Player Mode
- 🎨 UI with Gradients
- 🌟 Smooth Animations
- 📱 Responsive Design
- 🏆 Score Tracking
- 🔄 Quick Game Reset
- 📊 Game Statistics

## 🚀 Getting Started

### Prerequisites
- Flutter (latest version)
- Dart SDK
- Android Studio / VS Code

### Installation
```bash
# Clone repository
git clone https://github.com/yourusername/modern_tictactoe.git

# Navigate to project
cd modern_tictactoe

# Install dependencies
flutter pub get

# Run app
flutter run
📁 Project Structure

🎮 How to Play
Choose game mode (1P or 2P)
Player X starts first
Tap empty cells to place marks
Get three in a row to win
Use 'New Game' to restart
